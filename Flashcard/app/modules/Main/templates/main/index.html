Home

Welcome to the Flashcard App. Here you can create, study, and manage your flashcards.

Instructions

  • Create a new flashcard set by clicking on the create page
  • Add flashcards to your set by entering a question and answer for each card.
  • Save your flashcard set once you have added all your cards.
  • Alternatively you can import your flashcards using a .csv or excel (.xls or .xlsl) file
  • Study your flashcards by selecting a set and using the "Study" mode.
  • Manage your flashcard sets by editing or deleting them as needed.
  • You can only edit the name and description of the packs as of now
